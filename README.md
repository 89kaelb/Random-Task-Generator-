# Random Task Generator

**Автор:** Калашников Григорий Андреевич
**Описание:** Простое GUI-приложение для генерации случайных задач по категориям (учёба, спорт, работа) с сохранением истории и фильтрацией.
**Технологии:** Python 3, Tkinter, JSON, Git.

## Возможности:
- Генерация случайной задачи из предопределённого списка.
- Фильтрация задач по типу.
- История генераций с сохранением в файл `tasks.json`.
- Очистка истории.
- Проверка корректности ввода (встроена в логику).
